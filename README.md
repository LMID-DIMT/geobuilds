# geobuilds

*In development*

## /scripts
A series of bash scripts that will install OSGeo projects (PostgreSQL/PostGIS, GeoServer) on a fresh CentOS 7 install.

### build-pgsql.sh
```
$ cd scripts
$ sudo chmod +x build-pgsql.sh
$ ./build-pgsql.sh
```

### build-geoserver.sh
```
$ cd scripts
$ sudo chmod +x build-geoserver.sh
$ ./build-geoserver.sh
```
